## üÜò TROUBLESHOOTING

### Issue: API Errors
**Solution**: The system automatically retries up to 5 times. If persistent, check your internet connection.

### Issue: Slow Responses
**Solution**: Switch to Gemini 2.0 Flash model for faster responses, or reduce the length setting.

### Issue: Code Not Highlighting
**Solution**: Ensure code is wrapped in triple backticks with language identifier (e.g., ```python).

### Issue: Context Not Working
**Solution**: Use the CONTEXT button to verify message count. If needed, clear context and restart.
