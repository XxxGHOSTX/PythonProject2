- Check console for actual port number
- Manually navigate to `http://localhost:[PORT]/thalos_coding_agent.html`

### Issue: Module Not Found

**Solution**: Reinstall dependencies:

### Issue: Slow Generation

**Cause**: High complexity setting + large request
**Solution**: 
- Lower complexity slider (try 5-7)
- Break large requests into smaller parts
- Use specific mode instead of "Full Application"
